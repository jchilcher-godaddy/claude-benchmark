# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in claude-benchmark, please report it responsibly.

**Email:** Open a [GitHub Security Advisory](https://github.com/jchilcher/claude-benchmark/security/advisories/new) (preferred) or email the maintainer directly.

**Please do not** open a public GitHub issue for security vulnerabilities.

## What to include

- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

## Response

I'll acknowledge receipt within 72 hours and aim to release a fix within 30 days for confirmed vulnerabilities.

## Scope

This policy covers the `claude-benchmark` CLI tool and its Python source code. It does not cover third-party dependencies, though reports about vulnerable dependencies are welcome.
